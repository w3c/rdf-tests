RDF Test Suites

These test suites are a product previous RDF working groups, the RDF-star Working Group, and has been maintained by the RDF Test Curation Community Group.

Community maintained at https://github.com/w3c/rdf-tests/tree/main/rdf.

Conformance with RDF 1.1 specifications can be determined via successfully running the rdf11 tests for relevant specifications.

Conformance with RDF 1.2 specifications requires passing both the rdf11 tests and the rdf12 tests for relevant specifications.
